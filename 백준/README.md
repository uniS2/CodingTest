## 🤔 HOW? 진행방법

1. 목표: 최대 5분 내 풀이법 생각하기
2. 코테 테스트 통과 후 더 나은 문제 풀이법이 없나 고민하기
3. 필요한 경우 코드리뷰, 오답노트 정리하기
4. 스터디 진행: [단계별로 풀어보기](https://www.acmicpc.net/step) 한 주마다 최소 한 단계씩 풀기
